Instalación Dual Boot Windows 11 + Abalinux con grub2win

Abalinux (Maqueta Abalar Libre) es una distribución basada en Debian creada por la Xunta de Galicia. Aquí te explicamos cómo instalarla en dual boot con Windows 11 usando grub2win, una herramienta que permite gestionar GRUB2 directamente desde Windows.

Importante: Haz copias de seguridad antes de particionar el disco. Este proceso puede causar pérdida de datos si no se hace correctamente.
  1. Preparación
    • Descarga la ISO de Windows 11 desde Microsoft.
    • Descarga Abalinux desde espazoAbalar.
    • Crea USB bootables con Rufus.
    • Descarga grub2win desde SourceForge.
    • En BIOS/UEFI: Activa modo UEFI y desactiva Secure Boot temporalmente.
  2. Instala Windows 11 primero
    • Bootea desde el USB de Windows.
    • Crea partición NTFS para Windows y deja espacio libre sin asignar para Abalinux (mínimo 30 GB).
    • Completa la instalación.
  3. Instala Abalinux
    • Bootea desde el USB de Abalinux.
    • En el particionado, usa el espacio libre para crear particiones ext4 y swap.
    • Importante: No instales GRUB en el MBR/EFI principal. Si el instalador pregunta, elige instalar GRUB solo en la partición root de Linux o salta ese paso.
    • Finaliza la instalación.
  4. Configura el dual boot con grub2win
    • Arranca en Windows 11.
    • Ejecuta grub2win como administrador.
    • La herramienta detectará automáticamente la instalación de Abalinux.
    • Configura el menú de arranque y aplica los cambios.
    • Reinicia: ahora verás el menú de GRUB gestionado por grub2win con opciones para Windows y Abalinux.
  5. Solución de problemas
    • Si no detecta Linux, usa la opción manual para agregar la partición.
    • Asegúrate de que ambas instalaciones sean UEFI.

Quitar restricciones DNS en ordenadores Abalar (sin permisos administrativos)

Los ordenadores del proyecto Abalar incluyen filtros DNS para bloquear contenidos no adecuados en entornos educativos. Si no tienes contraseña de administrador, puedes usar la consola de GRUB para obtener acceso root temporal antes de que cargue el sistema.

Advertencia: Modificar las restricciones puede ir contra las normas del centro educativo. Úsalo bajo tu propia responsabilidad.
  1. Accede a la consola de GRUB antes del arranque
    • Enciende el ordenador.
    • Si no aparece el menú GRUB, mantén pulsada la tecla Shift durante el arranque.
    • Selecciona la entrada de Abalinux y pulsa e para editar.
    • Busca la línea que empieza por linux y al final añade: init=/bin/sh
    • Pulsa Ctrl+X o F10 para arrancar.
    • Aparecerás en una consola con permisos root sin necesidad de contraseña.
  2. Modifica los DNS
    • Haz el sistema escribible: mount -o remount,rw /
    • Edita el archivo DNS: nano /etc/resolv.conf
    • Borra las líneas existentes y añade: 
      nameserver 8.8.8.8
nameserver 8.8.4.4
# O usa Cloudflare:
# nameserver 1.1.1.1
# nameserver 1.0.0.1
    • Guarda (Ctrl+O → Enter → Ctrl+X).
    • Para que no se revierta fácilmente: chattr +i /etc/resolv.conf
  3. Desactiva servicios de filtrado (opcional)
    • Busca servicios relacionados con filtro y desactívalos (ejemplo genérico):
      systemctl disable --now nombre-del-servicio-filtro
  4. Continúa el arranque
    • Escribe exec /sbin/init para continuar el inicio normal del sistema.
    • O reinicia con reboot.
  5. Alternativas
    • Bootea desde un USB Live de otra distribución Linux y edita los archivos montando la partición.
    • Usa una VPN una vez dentro del sistema (si permite instalar software).
